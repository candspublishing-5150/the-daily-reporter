# The Daily Reporter

> Connecting California contractors to construction bidding leads since 1994.

The Daily Reporter (TDR) is a certified California trade newspaper published by C&S Publishing, a woman-owned small family business (DBE/WBE/SBE/SMBE certified). It serves as the go-to source for public construction bidding leads, outreach advertising, and prime-to-sub outreach across California.

## What TDR Does

- Publishes multiple weekly editions of the newspaper (Southern CA, Northern CA, CA State Water, Builder's Blueprint)
- Lists public jobs open for bid, sorted by county
- Runs legal "good-faith" outreach advertisements (certified by CA Dept. of General Services since 1994)
- Emails bidding leads and prime-seeking-sub notifications to 13,000+ California construction firms
- Offers TDR Plus — a premium subscription service for enhanced lead access

## This Rewrite

This repository is a ground-up rewrite of [thedailyreporter.net](https://thedailyreporter.net) using a modern stack.

### Tech Stack

| Layer | Technology |

| CI/CD | [GitHub Actions](https://github.com/features/actions) |
